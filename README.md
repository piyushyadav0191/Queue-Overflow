# Queue-Overflow

A place where you can ask questions and give answers

# Tech used

- Nextjs 13
- Server Actions
- Node
- MongoDB
- clerk
- Web hooks

# Current feature

- Ask Question
- Give your Answer'
- Upvote/Downvote Question
- Save question
- Markdown and code highlighting
- Local search
- Global Search
- Filterization
- Reputation algo

# Future features

- Pagination
- Generate AI answer

## Getting Started

First, run the development server:

```
git clone https://github.com/piyushyadav0191/Queue-Overflow
cd Queue-Overflow
npm i
copy .env.example variables and paste in new .env with your credentials
npx prisma db push
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
